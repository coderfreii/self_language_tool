# volar-service-sass-formatter

Volar plugin for [Sass Formatter](https://sass-formatter.syler.de/).

## Installation

```sh
npm install volar-service-sass-formatter
```

## Usage

`volar.config.js`

```js
module.exports = {
	services: [
		require('volar-service-sass-formatter').create(),
	],
};
```
